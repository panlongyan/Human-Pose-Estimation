# 2. 速度/加速度归一化

## 2.1 目的

速度/加速度归一化的目标是：

- 消除人体整体运动（global motion）的影响；
- 去除骨盆平移带来的共同加速度；
- 消除人体朝向变化的影响；
- 获得各传感器相对于骨盆的局部运动信息。

经过归一化后，相同动作在不同朝向下具有相似的输入表示，从而提高模型的泛化能力。

---

## 2.2 处理流程

### 2.2.1 转换到标定时刻 SMPL 身体坐标系

在本文中，旋转矩阵的上下标约定为：

```math
R_{A}^{B}
```

表示从右下标坐标系 $A$ 变换到右上标坐标系 $B$ 的旋转矩阵，即：

```math
A \rightarrow B
```

在初始标定阶段，通过标定获得每个传感器坐标系到 SMPL 身体坐标系的旋转矩阵：

```math
R_{S}^{T}
```

其中：

- $S$ 表示传感器坐标系（Sensor coordinate system）；
- $T$ 表示标定时刻 SMPL 身体坐标系（Template SMPL coordinate system）。

利用该旋转矩阵，将传感器加速度转换到标定时刻 SMPL 身体坐标系：

```math
a_{s}^{T}(t)=R_{S}^{T}a_{s}^{S}(t)
```

其中：

- $a_{s}^{S}(t)$：传感器 $s$ 在传感器坐标系中的加速度；
- $R_{S}^{T}$：传感器坐标系到标定时刻 SMPL 身体坐标系的旋转矩阵；
- $a_{s}^{T}(t)$：传感器 $s$ 的加速度在标定时刻 SMPL 坐标系中的表示。

经过该步骤，所有传感器数据均转换到同一个参考坐标系：

```math
\text{Template SMPL coordinate system}
```

---

### 2.2.2 减去 root（骨盆）加速度，去除整体运动

人体运动过程中，骨盆作为人体整体运动的参考点。

定义：

```math
a_{root}^{T}(t)
```

表示 root（骨盆）传感器在标定时刻 SMPL 坐标系中的加速度。

对于任意传感器 $s$，计算其相对于骨盆的加速度：

```math
\Delta a_{s}^{T}(t)=a_{s}^{T}(t)-a_{root}^{T}(t)
```

其中：

- $a_{s}^{T}(t)$：传感器 $s$ 在标定时刻 SMPL 坐标系中的加速度；
- $a_{root}^{T}(t)$：骨盆 root 在标定时刻 SMPL 坐标系中的加速度；
- $\Delta a_{s}^{T}(t)$：去除整体运动后的局部加速度。

该步骤去除了：

```math
\text{Global translation motion}
```

保留了：

```math
\text{Relative body motion}
```

---

### 2.2.3 转换到当前 root（骨盆）坐标系

由于人体运动过程中会改变朝向，需要进一步消除人体朝向变化的影响。

定义：

```math
R_{root}^{T}(t)
```

表示从当前时刻 root（骨盆）坐标系变换到标定时刻 SMPL 身体坐标系的旋转矩阵，即：

```math
\text{Current root coordinate system}
\rightarrow
\text{Template SMPL coordinate system}
```

其逆矩阵：

```math
(R_{root}^{T}(t))^{-1}
```

等价于：

```math
R_{T}^{root}(t)
```

用于完成如下坐标转换：

```math
\text{Template SMPL coordinate system}
\rightarrow
\text{Current root coordinate system}
```

最终归一化后的加速度为：

```math
\bar{a}_{s}(t)=R_{T}^{root}(t)\Delta a_{s}^{T}(t)
```

展开后得到：

```math
\bar{a}_{s}(t)=R_{T}^{root}(t)(a_{s}^{T}(t)-a_{root}^{T}(t))
```

---

## 2.3 最终结果含义

最终得到：

```math
\bar{a}_{s}(t)
```

其含义为：

传感器 $s$ 相对于骨盆 root 的局部加速度，并且表达在当前人体朝向坐标系中。

因此，该数据具有以下特点：

- 去除了人体整体平移产生的影响；
- 消除了人体朝向变化带来的影响；
- 保留了人体各部位之间的相对运动信息。

最终输入模型的数据参考系为：

```math
\text{Current root coordinate system}
```

而不是：

```math
\text{Template SMPL coordinate system}
```

---

## 2.4 整体流程总结

整体流程可以表示为：

```math
a_{s}^{S}(t)
\rightarrow
a_{s}^{T}(t)
\rightarrow
\Delta a_{s}^{T}(t)
\rightarrow
\bar{a}_{s}(t)
```

对应步骤为：

1. 从传感器坐标系获得原始加速度；
2. 转换到标定时刻 SMPL 身体坐标系；
3. 减去 root（骨盆）加速度，去除整体运动；
4. 转换到当前 root 坐标系；
5. 得到归一化后的局部加速度。
